## Run Locally

Clone the project

```bash
  git clone https://github.com/iamkavinnnn/CLI-app-in-rust.git
```

Go to the project directory

```bash
  cd cli-app
```

Install required dependencies in cargo.toml file (if not present)


Build and run the project

```bash
  cargo run
```
